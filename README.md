Portfolio Site 


## Tech Stack

**Client:** React, sass, react-responsive-carousel




## Run Locally

Clone the project

```bash
  git clone https://github.com/Vivek7038/my-portfolio-site.git
```

Go to the project directory

```bash
  cd my-portfolio-site

```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
